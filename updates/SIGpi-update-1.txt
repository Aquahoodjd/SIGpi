Welcome to SIGpi-update

Given a SigPi Release 1 install, this script updates the installation to Release 2

Installation will continue with the following steps:

- Choose Digital Decoders (recommend all)
- Choose if you want to update Amateur Radio Applications to latest-compiled releases
- Choose Additional Tools
- Perform apt update and upgrade
- Check existing and install new dependencies
- Remove GNUradio 3.7 and compile/build/install GNuradio 3.8
- Install your choices
- Update Menus

Choosing to update Amateur Radio Applications to latest-compiled releases can add an hour
to update time. Only choose this option if you have a specific need only addressed by latest version
You were warned.
